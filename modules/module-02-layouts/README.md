# Module 02 — Basic Layouts & Responsive Design

> Build screens that adapt cleanly from a phone to a foldable, tablet, and desktop.

**Status:** ✅ **Fully written** — every lesson meets the [Authoring Guide](../../AUTHORING-GUIDE.md) standard.
**Prerequisites:** [Module 01](../module-01-introduction/README.md).
**Level:** 🟢🟡 · **Est.:** 6–8 hrs

## What you'll be able to do
- Compose layouts with `Row`/`Column`/`Box` and arrangement/alignment fluently.
- Build efficient lists and grids with `LazyColumn`/`LazyGrid` (keys + content types).
- Structure a screen with `Scaffold`.
- Make a UI adaptive with **Window Size Classes** across foldables/tablets/desktop.
