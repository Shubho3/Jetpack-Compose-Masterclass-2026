# Module 10 — Animations Masterclass

> Pick the right animation API for each job and build shared-element navigation that feels native.

**Status:** ✅ **Fully written**
**Prerequisites:** [Module 06 — Side Effects](../module-06-side-effects/README.md).
**Level:** 🟡🔴 · **Est.:** 7–9 hrs

## What you'll be able to do
- Choose between value, visibility, content, and gesture-driven animations.
- Coordinate multiple animations with a `Transition`.
- Implement shared-element transitions and use `LookaheadScope`.
