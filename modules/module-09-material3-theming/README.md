# Module 09 — Material 3 Theming

> Build a coherent, themable design system with dynamic color, typography, and shapes.

**Status:** ✅ **Fully written**
**Prerequisites:** [Module 02](../module-02-layouts/README.md).
**Level:** 🟢🟡 · **Est.:** 4–6 hrs

## What you'll be able to do
- Apply the Material 3 theming model (color/type/shape) correctly.
- Use **color roles** instead of hardcoded colors.
- Wire up **dynamic color** (Material You) with sensible fallbacks.

## Project
A complete theme (light/dark/dynamic) + a component gallery screen.

## Interview focus
Why color roles beat raw colors; dynamic color fallback; theming for accessibility/contrast.

## AI assistant focus
Generating a color scheme from a brand color; reviewing for hardcoded colors and missing dark variants.
